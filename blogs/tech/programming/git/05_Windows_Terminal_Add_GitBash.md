# Windows Terminal 添加 GitBash.md

Windows Terminal 设置 Json 中加入

```json
"list": 
[
	{
        "commandline": "D:\\Program Files\\Git\\bin\\bash.exe",
        "guid": "{11a66a21-0c86-4522-aa0d-415cce5f172c}",
        "hidden": false,
        "icon": "D:\\Program Files\\Git\\mingw64\\share\\git\\git-for-windows.ico",
        "name": "Git Bash",
        "startingDirectory": "%UserProfile%",
        "useAcrylic": true
    }
]
```
