让多个版本的Node.js共存



使用 [NVM (Node.js Version Management)](https://github.com/coreybutler/nvm-windows)，即可管理多个 Node.js 版本



安装完成后，使用 `nvm ls` 即可查看已安装的 Node.js 版本

```bash
perry@P-Elitebook19 MINGW64 ~
$ nvm ls

  * 14.17.1 (Currently using 64-bit executable)
```



使用 `nvm install [version]` 可安装其他版本 Node.js，可用列表见 https://nodejs.org/dist/index.json



安装完成后，即可看到两个版本

```bash
perry@P-Elitebook19 MINGW64 ~
$ nvm ls

    18.6.0
  * 14.17.1 (Currently using 64-bit executable)
```



使用 `nvm use [version]` 即可切换 Node.js 版本 

```bash
perry@P-Elitebook19 MINGW64 ~
$ nvm use 18.6.0
Now using node v18.6.0 (64-bit)

perry@P-Elitebook19 MINGW64 ~
$ nvm ls

  * 18.6.0 (Currently using 64-bit executable)
    14.17.1
```
