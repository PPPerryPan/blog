安装 Node.js

```
node -v
```

全局安装 cli

```
 npm install -g @vue/cli
```

全局安装 cli-service-global

```
npm install -g @vue/cli-service-global
```





在原生 JS 开发中，导入模块通常用 `<script src="..."></script>`，

如： `<script src="https://cdn.jsdelivr.net/npm/axios/dist/axios.min.js"></script>`



在单文件组件开发中：`npm install ...` 或 `npm i ...`，

如： `npm i axios`

然后在项目中导入： `import axios from 'axios'`
